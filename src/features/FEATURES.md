# Features Folder Guide

Each folder in `src/features` represents one route-level page module. The page composes reusable components and workflow data for one operational stage.

- dashboard: overview metrics and high-level visuals
- queue: token reception and queue intake
- patient: patient lookup and auto-fill
- orders: order reception + test selection
- barcode: barcode preview and print controls
- sample: sample collection confirmation flow
- processing: queue table and category filters
- results: dynamic result entry form
- review: senior review actions
- approval: final verification and publishing
- delivery: report dispatch controls
- notifications: role alerts center
- analytics: chart-heavy reporting
- staff: role performance cards
- profile: profile view + edit
- settings: panel configuration placeholders
- activity: timeline and audit-like feed
- mobile-quick-actions: mobile first launcher
